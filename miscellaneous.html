Miscellaneous API¶

Release:0.8.0
flacmanager.get_config()[source]¶

Return the configuration settings.

Returns:settings read from flacmanager.ini
Return type:configparser.ConfigParser

The configuration is initialized with default/empty values and saved to disk if it does not exist.

flacmanager.save_config()[source]¶

Write the configuration settings to an INI-style file.

flacmanager.make_tempfile(suffix='.tmp', prefix='fm')[source]¶

Create a temporary file.

Parameters:
  • suffix (str) – the default file extenstion
  • prefix (str) – prepended to the beginning of the filename
Returns:

the temporary file name
Return type:

str

The temporary file will be deleted automatically when the program exits.
